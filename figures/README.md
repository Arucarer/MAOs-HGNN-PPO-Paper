# Figures

Store publication-ready figures in this folder. Prefer vector PDF for diagrams and plots; use high-resolution PNG only for raster content.

Suggested filenames:

- `problem_environment.pdf`
- `heterogeneous_graph_schema.pdf`
- `hgnn_ppo_framework.pdf`
- `training_curve.pdf`
- `ablation_results.pdf`
- `generalization_results.pdf`
- `gantt_chart.pdf`

Every figure must be introduced and interpreted in the text, use readable fonts at final print size, and have a self-contained caption.
